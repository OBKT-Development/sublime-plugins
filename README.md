###Sublime Text 3 設定

---

####導入方法

Sublime Text 3の設定ファイル、パッケージの場所は以下です。

```
Users/ライブラリ/Application Support/Sublime Text3/Packages
```

この中に、Packagesフォルダ以下を丸々突っ込むといいです。

反映されない場合はSublime Text 3を再起動すると良いです。


1.ターミナルで移動すると楽です。

```
$cd Library/Application\ Support/Sublime\ Text\ 3/Packages
```

2.操作しやすいようにFinderで開く。

```
$open .
```

####プラグイン一覧

すっごい便利、ほぼ必須

```
■Emmet
入力の手間が減ります。絶対必要。
https://packagecontrol.io/packages/Emmet

■Emmet Css Snippets
入力の手間が減ります。CSS。
https://packagecontrol.io/packages/Emmet%20Css%20Snippets

■AutoFileName
ファイルパスとか予測で入力候補にあげてくれます。便利です。
https://packagecontrol.io/packages/AutoFileName

■HalfImageSize
Ctrl + Shift + h でimgタグのwidthとheightが半分になる。とっても便利。
https://github.com/Satoh-D/HalfImageSize

■Sass
Sassのハイライト使えます。Sass便利。
https://packagecontrol.io/packages/Sass

■SassBeautify
Sassが綺麗に書けます。css設計が上手くなるわけではないです。
https://packagecontrol.io/packages/SassBeautify

■SCSS
Sass使う為に入れないといけなかったはずです。多分。
https://packagecontrol.io/packages/SCSS
```

それなりに便利、あると捗る。

```
■InputSequence
Ctrl + Shift + 0 で連番振ってくれる。とっても便利。でも、すぐ忘れる。
https://github.com/kazu1107/InputSequence

■LiveStyle
使えるような使えないような微妙なのです。Chromeのプラグインと連動。絶対配置しまくりページには効果的。
https://packagecontrol.io/packages/LiveStyle
```

```

■Better CoffeeScript

https://packagecontrol.io/packages/Better%20CoffeeScript

■CSScomb
CSSファイルの整形してくれるらしいです。使った事ないです。
https://packagecontrol.io/packages/CSScomb

■LESS

https://packagecontrol.io/packages/LESS
```
